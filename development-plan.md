# MBTI 性格测试小程序 — 开发计划

## 项目概述

微信小程序，用户通过答题测出 MBTI 16 型人格，生成可视化结果卡片，支持社交分享。

**目标用户**：18-35 岁年轻群体，对自我认知/社交标签有兴趣  
**核心卖点**：结果可视化精美 + 社交分享裂变 + 轻量速测

---

## 技术选型

| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | 微信小程序原生 | 包体小、性能好、无额外依赖 |
| 语言 | TypeScript | 类型安全，维护性好 |
| 样式 | WXSS + CSS 变量 | 原生方案，够用 |
| 状态管理 | 自行实现轻量 Store | 小程序体量不需要 MobX/Redux |
| 图表 | Canvas 2D 自绘 | 结果雷达图/维度图，不引库减少包体 |
| 数据存储 | 云开发（CloudBase） | 免服务器，自带数据库和云函数 |

---

## MBTI 题目体系

4 个维度，每维度 7 题，共 28 题（速测版），约 3 分钟完成：

| 维度 | 对立极 | 含义 |
|------|--------|------|
| E / I | 外向 vs 内向 | 能量来源 |
| S / N | 感觉 vs 直觉 | 信息获取 |
| T / F | 思维 vs 情感 | 决策方式 |
| J / P | 判断 vs 知觉 | 生活方式 |

每题 2 选项，选 A 加左极 1 分，选 B 加右极 1 分，取高分侧为该维度结果，4 维度组合得 16 型之一。

## 功能模块

### P0 — MVP（必须）

1. **首页**：品牌视觉 + 开始测试入口
2. **答题流程**：逐题呈现，进度条，滑动切换，支持回退
3. **计分引擎**：4 维度独立计分 → 组合 16 型
4. **结果页**：
   - 性格类型名称 + 代号（如 INTJ 建筑师）
   - 4 维度雷达图 / 柱状图
   - 性格描述（3-5 段核心特质）
   - 保存结果到本地 + 云端
5. **分享卡片**：Canvas 绘制分享图，含头像/昵称/类型/二维码

### P1 — 增强（MVP 后迭代）

6. **历史记录**：查看过往测试结果，对比变化
7. **16 型百科**：浏览所有类型详情
8. **匹配度**：两个类型之间的社交/工作匹配度
9. **深度测试**：60 题完整版，更精确

### P2 — 社交裂变（增长期）

10. **好友对比**：分享链接，双方结果对比
11. **排行榜/地图**：各类型人数统计热力图
12. **类型社群**：同类型用户聚合（轻论坛）

## 视觉风格

- **风格**：Apple Design Language — 毛玻璃 + 浅色系
- **底色**：纯白 #FFFFFF / 浅灰 #F5F5F7
- **毛玻璃**：backdrop-filter: blur(40px) saturate(180%)，白色半透明背景 rgba(255,255,255,0.72)
- **主色**：系统蓝 #007AFF，辅色柔和紫 #AF52DE / 绿 #34C759 / 橙 #FF9500
- **16 型各代表色**：取 Apple 色板，低饱和度柔色
- **圆角**：大圆角 16-20px，卡片感
- **阴影**：极轻投影 box-shadow: 0 2px 20px rgba(0,0,0,0.06)
- **字体**：系统默认（-apple-system），粗细对比 700/400
- **留白**：大量负空间，呼吸感
- **动画**：弹性缓动，淡入淡出，无夸张动效
- **分割**：细线 0.5px rgba(0,0,0,0.08)，或无边框纯留白分割

## 风险点

1. **审核**：MBTI 涉及心理测试，避免"诊断"措辞，用"趣味参考"定位
2. **同质化**：市面 MBTI 小程序多，视觉和分享体验是差异化关键
3. **数据**：题目和描述需要原创或授权，不能直接搬运
